# 💧 AquaTrack Pro — Умный трекер воды

PWA-приложение для отслеживания водного баланса с персональными рекомендациями.

## 🚀 Возможности

### Основные функции
- ✅ **Трекер воды** — отслеживание ежедневного потребления
- ✅ **Калькулятор нормы** — расчёт на основе веса, роста, возраста, пола и активности
- ✅ **Типы напитков** — 5 видов с коэффициентами гидратации
- ✅ **Push-уведомления** — напоминания о питьё
- ✅ **Режим "Не беспокоить"** — авто-отключение уведомлений ночью

### Аналитика
- ✅ **Календарь целей** — визуальное отображение выполнения
- ✅ **Почасовой график** — статистика по часам
- ✅ **История записей** — полный список с типами напитков

### Геймификация
- ✅ **15 достижений** — мотивация для регулярного использования
- ✅ **Серии (streaks)** — подсчёт дней подряд
- ✅ **Конфетти** — празднование достижений

### Персонализация
- ✅ **Быстрые пресеты** — сохранение любимых объёмов
- ✅ **Советы дня** — 15 персонализированных рекомендаций
- ✅ **Экспорт данных** — выгрузка в JSON

## 📊 Формула расчёта нормы

Используется формула **Mifflin-St Jeor** для расчёта BMR + корректировка на активность:

```
BMR (мужчины) = 10 × вес + 6.25 × рост - 5 × возраст + 5
BMR (женщины) = 10 × вес + 6.25 × рост - 5 × возраст - 161

Норма воды = 35мл × вес × коэффициент активности
```

## 🔔 Уведомления

Приложение поддерживает push-уведомления через Service Worker API.

### Расписание напоминаний
- **Утро**: 08:00 (настраивается)
- **День**: 13:00 (настраивается)
- **Вечер**: 18:00 (настраивается)

### Режим "Не беспокоить"
По умолчанию: 22:00 – 07:00 (настраивается)

## 🎨 Дизайн-система

Используется яркая градиентная тема с фиолетово-розовыми акцентами:

```css
--gradient-1: #6366F1  /* Indigo */
--gradient-2: #8B5CF6  /* Purple */
--gradient-3: #D946EF  /* Fuchsia */
--gradient-4: #EC4899  /* Pink */
```

## 📈 Статистика

- **Цель**: настраивается (500-5000 мл)
- **Записи**: неограниченно
- **История**: хранится локально

## 🔒 Конфиденциальность

Все данные хранятся локально в браузере (LocalStorage). Никакие данные не отправляются на сервер.
